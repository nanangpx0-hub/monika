# QUICK REFERENCE - ADMINISTRATOR
*Sistem MONIKA*

## 🔑 AKSES PENUH
✅ Semua menu dan fitur tersedia

## 📊 DASHBOARD
- **Total Dokumen**: Jumlah keseluruhan dokumen
- **Sudah Entry**: Dokumen selesai diproses  
- **Error/Anomali**: Dokumen bermasalah
- **Ranking Error**: Top 5 PCL dengan error terbanyak

## 🎯 TUGAS UTAMA

### 1. Kelola Kegiatan Survei
**Menu:** Master Kegiatan
- Buat kegiatan baru
- Atur status Aktif/Selesai
- Kode kegiatan harus unik

### 2. Monitor Performa Tim
**Menu:** Monitoring & Evaluasi
- Cek statistik real-time
- Analisis trend error
- Identifikasi bottleneck

### 3. Review Laporan
**Menu:** Laporan
- **Kinerja PCL**: Produktivitas & kualitas
- **Kinerja Pengolahan**: Efisiensi processing

## ⚡ QUICK ACTIONS
- **Tambah Kegiatan**: Master Kegiatan → Tambah
- **Lihat Error Detail**: Dokumen Survei → Filter Error
- **Export Data**: Screenshot/copy manual (fitur export belum ada)

## 🚨 ALERT INDICATORS
- Error rate > 10%: Perlu training PCL
- Backlog > 100 dokumen: Tambah resource pengolahan
- Tidak ada kegiatan aktif: Buat kegiatan baru
