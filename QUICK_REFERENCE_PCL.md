# QUICK REFERENCE - PCL (PETUGAS PENDATAAN)
*Sistem MONIKA*

## 🔑 AKSES TERBATAS
✅ Dashboard (view only)  
✅ Setor dokumen  
✅ Lihat status dokumen sendiri  
❌ Proses dokumen orang lain  

## 📊 DASHBOARD
- Statistik umum (informasi saja)
- Tidak bisa lihat detail pengguna lain

## 🎯 TUGAS UTAMA

### 1. Setor Dokumen
**Menu:** Dokumen Survei → Setor Dokumen
1. Pilih kegiatan aktif
2. Input kode wilayah
3. Pilih tanggal setor
4. Simpan

### 2. Monitor Status Dokumen
**Menu:** Dokumen Survei
- Cek status: Uploaded → Sudah Entry/Error/Valid
- Catat ID dokumen untuk tracking

## 📋 STATUS DOKUMEN
- 🔵 **Uploaded**: Menunggu diproses
- 🟢 **Sudah Entry**: Selesai, tidak ada masalah
- 🔴 **Error**: Ada masalah, perlu perbaikan
- ✅ **Valid**: Sudah diperbaiki dan diterima

## ⚡ QUICK ACTIONS
- **Setor Dokumen**: Dokumen Survei → Setor Dokumen
- **Cek Status**: Dokumen Survei → Lihat tabel
- **Follow Up Error**: Hubungi petugas pengolahan

## 🚨 ALERT INDICATORS
- Status "Error": Segera perbaiki dokumen
- Dokumen lama status "Uploaded": Follow up ke pengolahan

## ✅ BEST PRACTICES
1. Cek kualitas dokumen sebelum setor
2. Catat ID dokumen
3. Follow up dokumen error segera
4. Gunakan kode wilayah konsisten

## 📞 ESCALATION
Dokumen error → Petugas Pengolahan → PML → Administrator
